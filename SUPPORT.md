# Support

## Project Status

This is an **experimental project** exploring integration between the Amplifier framework and the GitHub Copilot CLI SDK. We're sharing it openly to enable community learning and experimentation.

## Getting Help

- **Questions**: Feel free to open a [GitHub Discussion](../../discussions) for questions or ideas
- **Bug Reports**: If you encounter issues, please open a [GitHub Issue](../../issues) with reproduction steps
- **Documentation**: See [README.md](README.md) for setup instructions and usage examples

## What to Expect

As an experimental project:

- Response times may vary
- Breaking changes may occur without deprecation periods
- Features may be added, changed, or removed based on learnings

We appreciate your interest and patience as we explore this space together.

## Microsoft Support Policy

This project is not covered by Microsoft support plans or services. For security issues, please see [SECURITY.md](SECURITY.md).
